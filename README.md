# Hi there! 👋 I'm Harshavardhan Talap

Welcome to my GitHub! I am a third-year B.E. Electronics and Telecommunication (EnTC) student at DYPCOE Akurdi and a Google Student Ambassador. Coming from a diploma background, I enjoy bridging the gap between hardware and software, building projects that range from web and mobile applications to physical IoT devices.

### 🛠️ What I Do
* **App Development:** Building Android applications like *PlayBox* and *Expense Tracker* using Java.
* **Web Development:** Developing applications using the MERN stack and Python with Django.
* **IoT & Hardware:** Designing electronics, robotics, and automation systems (like my Smart Agriculture Monitoring Robot and custom Home Automation projects) using Arduino and other microcontrollers.
